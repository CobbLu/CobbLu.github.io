    定位

    Cobb_Lu

    定位 = 定位模式 + 边偏移

    注意：所有定位会提高层级压住标准流/浮动盒子

    # 1.定位模式

    # 1.1.relative相对定位

    1、相对于自身原来的位置定位

    2、不脱标保持原来的位置

    # 1.2.absolute绝对定位

    注意：绝对定位(固定定位)会压住下面标准流所有的内容

    1、相对于祖先元素来定位

    2、没有祖先元素或祖先元素未设置定位则以浏览器定位

    3、如果祖先元素有定位(相对、绝对、固定定位) , 则以最近一级的有定位祖先元素为参 考点移动位置

    4、脱标不保持原来的位置

    绝对定位盒子居中

    1、绝对定位盒子不能用margin: 0 auto;

    2、左右走50%left/right: 50%;

    3、内边距（负值）走自己盒子宽高的一半margin-left/bottom:

    <style>
	* {
		padding: 0;
		margin: 0;
	}
	.content {
		width: 120px;
		height: 100px;
		background-color: blue;
		position: absolute;
		left: 50%;
		top: 50%;
		margin-left: -50px;
		margin-top: -50px;
	}
</style>
<body>
  <div class="content"></div>
</body>

    # 1.3.fixed固定定位

    1、相对于浏览器的可视区域来定位

    2、与父元素无关，不随滚动条滚动

    3、脱标不保持原来的位置

    固定”回到顶部“按钮布局

    1、让按钮固定定位走left:50%;，走到浏览器可视区的一半

    2、让按钮margin-left，再走版心宽度的一半

    <style>
	.content {
		width: 1200px;
		height: 1500px;
		background-color: red;
		margin: 0 auto;
	}
	.backTop {
		position: fixed;
		left: 50%;
		bottom: 10%;
		margin-left: 600px;
		width: 50px;
		height: 60px;
		background-color: blue;
	}
</style>
<body>
  <div class="backTop"></div>
  <div class="content"></div>
</body>

    # 1.4.sticky粘性定位

    兼容性差

    1、以浏览器的可视窗口为参照点移动元素(固定定位特点)

    2、粘性定位**占有原先的位置(**相对定位特点)

    3、必须添加top、left、 right、 bottom 其中一个才有效

    <style>
	.content {
		width: 1200px;
		height: 10000px;
	}
	.box {
		position: sticky;
		margin: 200px auto;
		/* 滑动到最高点时box固定 */
		top: 0;
		width: 800px;
		height: 60px;
		background-color: blue;
	}
</style>
<body>
	<div class="box"></div>
	<div class="content"></div>
</body>

    # 3.z-index定位叠放

    控制盒子前后顺序

    z-index: $;

    1、数值越大盒子越靠上

    2、数字后面不能加单位

    3、只有定位的盒子才有z-index属性
