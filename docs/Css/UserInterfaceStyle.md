## 1.鼠标样式

*在元素上移动的鼠标指针采用何种系统预定义的光标形状显示*

>cursor: value;

| 属性值      | 描述     |
| ----------- | -------- |
| default     | 默认光标 |
| pointer     | 手       |
| move        | 移动     |
| text        | 文本     |
| not-allowed | 禁止     |



## 2.焦点轮廓线

> outline: none;



## 3.防止拖拽文本域

>resize: none;
